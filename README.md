## 🔧 Topics Covered

### LPIC-1 (System Administration)
| Topic | Skills |
|-------|--------|
| Essential Commands | find, grep, sed, awk, pipes, redirection |
| File Management | permissions, ownership, ACLs, links |
| Package Management | apt, yum/dnf, dpkg, rpm, compilation |
| Devices & Filesystems | fdisk, mkfs, mount, LVM, RAID |
| Boot Process | GRUB2, systemd, init, runlevels |
| User Management | useradd, groups, PAM, sudo, /etc/shadow |
| Networking | ip, ss, nmcli, routing, DNS client |
| Security | iptables, SSH keys, GPG, file permissions |

### LPIC-2 (Network Professional)
| Topic | Skills |
|-------|--------|
| DNS | BIND9, zones, recursion, DNSSEC |
| Web Server | Apache, Nginx, virtual hosts, SSL/TLS |
| File Sharing | NFS, Samba, FTP (vsftpd) |
| Email | Postfix, Dovecot, spam filtering |
| DHCP | ISC DHCP, reservations, failover |
| VPN | OpenVPN, WireGuard, IPSec |
| Monitoring | Nagios, Prometheus, Grafana |

### LPIC-3 (Enterprise)
| Topic | Skills |
|-------|--------|
| LDAP | OpenLDAP, directory design, replication |
| Kerberos | Authentication, KDC, keytabs |
| High Availability | Pacemaker, DRBD, load balancing |
| Virtualization | KVM, libvirt, containers, Docker |
